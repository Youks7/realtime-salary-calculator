# 实时时薪计算器

## English Version: [README.en.md](./README.en.md)

一个用来记录当天工作时长、计算时薪和下班结果的小型静态网页。

在线访问：

https://youks7.github.io/realtime-salary-calculator/

GitHub 仓库：

https://github.com/Youks7/realtime-salary-calculator

## 项目简介

这个网站根据用户输入的月收入和合同工时，实时计算当天工作的时薪情况。

它的核心用途很直接：让你在工作开始后看到自己已经工作了多久、标准时薪是多少、当前真实时薪是多少，并在结束工作时生成当天结果。

网站是纯前端项目，不需要后端、数据库或账号系统。

## 主要功能

- 自定义输入月收入
- 自定义输入每日合同工时
- 点击“开始工作”后才开始计时
- 开始工作时自动打开 ChatGPT
- 支持暂停工作和继续工作
- 点击“开始摸鱼”时自动打开抖音
- 点击“结束工作”后显示当天总结
- 点击“新的一天”后重置计时
- 未开始工作时，工作时长保持为 `00:00:00`
- 未满合同工时前，当前真实时薪按合同工时计算，避免刚开始几秒出现离谱高时薪

## 计算逻辑

月平均计薪天数采用常见值：

```text
21.75 天
```

标准时薪：

```text
标准时薪 = 月收入 / 21.75 / 合同工时
```

当前真实时薪：

```text
未开始工作：0
未满合同工时：月收入 / 21.75 / 合同工时
超过合同工时：月收入 / 21.75 / 已工作小时
```

结束页的预估月薪：

```text
预估月薪 = 今日真实时薪 * 合同工时 * 21.75
```

## 使用方法

打开网站后：

1. 输入月收入。
2. 输入合同工时。
3. 点击“开始工作”。
4. 需要暂停时点击“暂停工作”。
5. 继续工作时点击“继续工作”。
6. 工作结束后点击“结束工作”查看当天总结。
7. 第二天使用时点击“新的一天”重置。

## 本地运行

这个项目可以直接双击 `index.html` 打开。

如果浏览器对本地文件弹窗限制比较严格，也可以在项目目录启动一个本地服务：

```powershell
python -m http.server 4173
```

然后打开：

```text
http://127.0.0.1:4173/
```

## 注意事项

- 网站是公开访问的，知道链接的人都可以打开。
- 当前计时数据只保存在当前浏览器页面中，不会跨设备同步。
- 开始工作打开 ChatGPT、开始摸鱼打开抖音，依赖浏览器允许用户点击触发的新标签页。
- 如果浏览器拦截弹窗，需要手动允许该网站打开新标签页。
